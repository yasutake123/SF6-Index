# SF6-Index
ストリートファイター6のキャラクターデータおよび技情報を集約・管理するためのWebアプリケーションです。

## 🎯 目標
### 第１目標
- 各キャラクターの技リストを網羅したWebサイトの構築
### 第２目標（案）
- 確定反撃集
- コンボ集

## 🛠 技術スタック
- Backend: Java 17 / Spring Boot 3.5.14
- Build Tool: Maven
- Data Format: JSON
- Frontend: HTML / Thymeleaf (予定)
